MasterMind Gems

A reimagining of the classic board game MasterMind.

Technologies

HTML5, CSS3, JavaScript, Heroku

Details

Web-based gaming project. The aim of this project was to learn more about the HTML5 canvas property with regards to drawing multiple dynamic graphics.
Staying true to the original game, this web-based version is optimized to fit any screen on initial startup, giving the player a view of the entire board at once.
